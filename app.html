FIZZ BUZZ: A Coding Project

Fizz Buzz is a JavaScript project that loops over an array of ordered numbers and when the number is divisble by three it prints Fizz and when it is divisble by five it prints Buzz.

Instructions

Just click the FizzBuzz button to watch it go. If you want to see it Fizz and Buzz on a different set of numbers change the values in the boxes.
